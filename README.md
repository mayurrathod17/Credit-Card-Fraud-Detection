# Credit Card Fraud Detection 🚀

## 📌 Project Overview
This project detects fraudulent credit card transactions using Machine Learning.

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
  
## 📦 Dataset

The dataset used in this project is the **Credit Card Fraud Detection** dataset from Kaggle.

- 📊 **Source:** [Credit Card Fraud Detection - Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
- 🏦 **Provider:** Machine Learning Group - ULB (Université Libre de Bruxelles)
- 📁 **File:** creditcard.csv
- 📏 **Size:** 144 MB
- 🔢 **Rows:** 284,807 transactions
- 🎯 **Fraud Cases:** 492 (0.17% of total — highly imbalanced)

> ⚠️ **Note:** Due to the large file size, the dataset is **not included** in this repository.  
> Please download it directly from Kaggle using the link above.

### How to Download:
1. Go to the Kaggle link above
2. Click **"Download"** (requires free Kaggle account)
3. Place `creditcard.csv` in the root folder of the project
   
## ▶️ How to Run
1. Download dataset
2. Run notebook / script
